# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. 
Envie um arquivo compactado contendo: 
- o projeto sem o node_modules; 
- um arquivo PDF mostrando um passo a passo com o build e a execução do código. 
 
Orientações sobre o projeto: 
- Criar uma classe que permita criar “calculadoras”. Uma calculadora tem 4 operações básicas: soma, subtração, multiplicação e divisão. Toda operação básica envolve dois números e retorna o resultado da operação.
- Tomando como base a classe que cria “calculadoras”, crie duas novas classes: uma capaz de criar “calculadoras científicas” e uma capaz de criar “calculadoras financeiras”.
- Além das operações básicas, uma “calculadora científica” é capaz de realizar operações de potenciação e fatorial.
Uma operação de potenciação recebe dois números, e calcula “o primeiro elevado ao segundo” (ex: 4^2 = 16).
- Calcular o fatorial (!) de um número inteiro positivo consiste em determinar o produto de todos os números de 1 até o número informado. Por exemplo, o fatorial de 4, escreve-se 4!, é 4! = 4*3*2*1 = 24 e o fatorial de 5 é 5! = 5*4*3*2*1 = 120. É necessário validar a entrada de dados, permitindo apenas números inteiros positivos.
- Além das operações básicas, uma “calculadora financeira” pode calcular juros simples e juros compostos.
- O código deve criar uma calculadora científica e fazer todas as operações possíveis (básicas e específicas).
- O código deve, também, criar uma calculadora financeira e fazer todas as operações possíveis (básicas e específicas).
- Todo o código precisa estar fortemente tipado.
