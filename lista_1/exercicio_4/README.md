# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. 
Envie um arquivo compactado contendo: 
- o projeto sem o node_modules; 
- um arquivo PDF mostrando um passo a passo com o build e a execução do código. 
 
Orientações sobre o projeto: 
- Desenvolver uma função que recebe como parâmetro uma lista de funcionários de um squad e retorna o salário total desse squad. 
- Um funcionário é representado por 3 campos: um atributo para o nome dele; um atributo para o valor que ele recebe por hora; e um atributo para o número total de horas que ele trabalha. 
- O código deve conter, também, um “console.log” que chama a função desenvolvida, provando que ela está funcionando corretamente. 
- Todo o código precisa estar fortemente tipado.
