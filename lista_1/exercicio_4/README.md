# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. 
Envie um arquivo compactado contendo: 
i) o projeto sem o node_modules; 
ii) um arquivo PDF mostrando um passo a passo com o build e a execução do código. 
 
Orientações sobre o projeto: 
i) Desenvolver uma função que recebe como parâmetro uma lista de funcionários de um squad e retorna o salário total desse squad. 
ii) Um funcionário é representado por 3 campos: um atributo para o nome dele; um atributo para o valor que ele recebe por hora; e um atributo para o número total de horas que ele trabalha. 
iii) O código deve conter, também, um “console.log” que chama a função desenvolvida, provando que ela está funcionando corretamente. 
iv) Todo o código precisa estar fortemente tipado.
