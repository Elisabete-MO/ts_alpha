# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. 
Envie um arquivo compactado contendo: 
- o projeto sem o node_modules; 
- um arquivo PDF mostrando um passo a passo com o build e a execução do código. 
 
Orientações sobre o projeto: 
- Desenvolver uma função que recebe como parâmetro uma lista de pedidos. Após 3 segundos, a função deve imprimir, na saída do terminal, o id e o status do primeiro produto. Após mais 3 segundos, deve imprimir o id e o status do próximo pedido da lista. Esse comportamento deve se repetir para todos os pedidos. 
- Cada pedido é formado por dois atributos: id e status. 
- Obrigatoriamente, o status de um pedido está restrito aos valores: "pending", "confirmed", "preparing", "ready", "delivered". 
- Todo o código precisa estar fortemente tipado.
