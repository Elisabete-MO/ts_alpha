# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. Seu projeto deve conter uma função que conta palavras.
Considere que as palavras são separadas por espaços em branco. Sua função recebe como parâmetro uma string e retorna um inteiro que corresponde a quantidade de palavras daquela string. Envie um arquivo compactado contendo:

i. o projeto sem o node_modules; 
ii. um arquivo PDF mostrando um passo a passo com o build e a execução do código.
