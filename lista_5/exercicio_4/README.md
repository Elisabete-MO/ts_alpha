# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. 
Envie um arquivo compactado contendo: 
- o projeto sem o node_modules; 
- um arquivo PDF mostrando a aplicação sendo buildada e usada.
 
Orientações sobre o projeto: 
- Desenvolver uma página web que funcione como uma calculadora básica, onde o usuário possa inserir dois números e realizar as operações matemáticas simples: adição, subtração, multiplicação e divisão.
- Caso o usuário tente realizar uma operação utilizando valores inválidos, deve receber uma mensagem apropriada.
- A página deve ser esteticamente agradável.
- Todo o código precisa estar fortemente tipado.
- Utilize Webpack e outros módulos que julgar necessários para gerar e servir o bundle (pacote).
