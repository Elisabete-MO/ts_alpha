# ts_alpha

Crie um projeto em TypeScript, instalando as dependências que julgar necessárias. 
Envie um arquivo compactado contendo: 
- o projeto sem o node_modules; 
- um arquivo PDF mostrando um passo a passo com o build e a execução do código. 
 
Orientações sobre o projeto: 
- Desenvolva uma página de cadastro de usuário que solicita o nome, e-mail e senha. A página deve ser visualmente atraente. No Front-End do seu projeto, implemente a validação dos campos de entrada.
- O campo “Nome” deve aceitar apenas letras e espaços em branco, e ter no mínimo 4 caracteres (desconsiderando os espaços em branco).
- O campo “E-mail” deve seguir um padrão válido.
- O campo “Senha” deve conter pelo menos uma letra e um número, e ter no mínimo 8 caracteres de comprimento.
- Se o usuário inserir um ou mais campos que não estejam de acordo com os critérios especificados, deverá ser apresentada uma mensagem de erro adequada indicando o problema em cada campo.
Quando todos os dados informados estiverem dentro dos padrões exigidos, um modal deve ser ativado, exibindo uma mensagem de confirmação que informe ao usuário que seu cadastro foi realizado com sucesso.
- É importante ressaltar que este é um projeto exclusivamente de Front-End, portanto, o cadastro não será efetivamente armazenado em um banco de dados, nem é necessário implementar um Back-End.
Todo o código precisa estar fortemente tipado.
